# ConverToRGB565_DragAndDrop
Конвертер изображений должен ускорить процесс разработки GUI интерфейса виброкалибратора.

Конвертер автоматизирует процесс преобразования png и bmp изображений в массивы чисел используемые в ПО калибратора. 
Массивы представляют собой последовательность чисел в формате uint16_t с закодированным в них цветом пикселей в кодировке RGB565.
1) Приложение поддерживает функцию Drag&Drop;
2) В приложение можно одновремнно закинуть не менее 50 изображений;
3) Сгенерированные данные  сохраняются в текстовые файлы в формате *.c.
